## Installation

1. Open VS Code  
2. Go to the **Extensions** panel  
3. Search for `Java File Badges`  
4. Click **Install**  

---

## Usage

Once installed, badges appear automatically in the explorer next to `.java` files.  

### Customizing badges

Users can change badge symbols in **Settings**:

```json
{
  "javaFileBadges.classBadge": "C",
  "javaFileBadges.interfaceBadge": "I",
  "javaFileBadges.enumBadge": "E",
  "javaFileBadges.abstractBadge": "A",
  "javaFileBadges.recordBadge": "R",
  "javaFileBadges.annotationBadge": "@"
}
